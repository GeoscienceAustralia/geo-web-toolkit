gaGoogleLayer
directive in module gawebtoolkit.core

Description

Overview

gaGoogleLayer directive is used to create a Google map.

Usage

as element:
<ga-google-layer
       layer-type="{string|@}"
       visibility="{string|@}">
</ga-google-layer>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
layerTypestring@
  • Required. Specified Google maps layer type. Eg, Hybrid.
visibilitystring@
  • A boolean value ('true', 'false') which enables or disables visibility of the layer.
