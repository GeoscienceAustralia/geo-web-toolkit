gaMapControl
directive in module gawebtoolkit.core

Description

Overview

gaMapControl is a wrapper for a native map control

Usage

as element:
<ga-map-control
       map-control-name="{string|@}"
       map-control-id="{string|@}"
       control-options="{string|@}"
       [container-element-id="{string|@}"]
       [control-enabled="{number|@}"]>
</ga-map-control>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
mapControlNamestring@
  • The name of the control that is intended to be added to the map layer ##mapControlName acceptable values## The following values can be used as control name: <
    • permalink:
    • overviewmap:
    • scale:
    • scaleline:
    • panzoombar:
    • darwin.panzoombar:
    • mouseposition:
    • attribution:
    • measureline:
    • measurepolygon:
    • wmsgetfeatureinfo:
mapControlIdstring@
  • A string value to be allocated to the control as its ID
controlOptionsstring@
  • A string value that requests a setting to the specified control
containerElementId
(optional)
string@
  • The id of the layer that this control will be added to
controlEnabled
(optional)
number@
  • A boolean value ('true' or 'false') that toggles the control on/off
