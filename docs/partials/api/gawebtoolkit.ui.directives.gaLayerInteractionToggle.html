gaLayerInteractionToggle
directive in module gawebtoolkit.ui

Description

This control provides a button to toggle interaction with a layer on/off

Usage

as element:
<ga-layer-interaction-toggle
       toggle-icon-source="{string}"
       map-controller="{object}"
       controller-emit-event-name="{string}"
       toggle-off-callback="{function}"
       toggle-on-callback="{function()}"
       on-layer-click-call-back="{function}"
       layer-interaction-id="{string}">
</ga-layer-interaction-toggle>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
toggleIconSourcestring
  • A string value of toggle button's icon URL
mapControllerobject
  • controller instance for the map
controllerEmitEventNamestring
  • event that is fired that passes the controller
toggleOffCallbackfunction
  • Callback function to be called when toggle button is turns off
toggleOnCallbackfunction()
  • Callback function to be called when toggle button turns on
onLayerClickCallBackfunction
  • Callback function to be called when the map layer is clicked
layerInteractionIdstring
  • ID of the layer that this toggle button will interact with
