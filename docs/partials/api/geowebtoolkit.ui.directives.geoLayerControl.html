geoLayerControl
directive in module geowebtoolkit.ui

Description

A control for turning on/off layers via provided '=mapController' as well as opacity slider

Usage

as element:
<geo-layer-control
       layers-data="{Layer[]}"
       map-controller="{mapController[]}"
       on-visible="{string}"
       on-hidden="{string}"
       on-opacity-change="{string}">
</geo-layer-control>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
layersDataLayer[]
  • Layers that the toolbar will interact with structure expected is a minimum of: {
       id: //unique id of the layer
   opacity: //value betweeb 0-100 representing the percentage of opacity of the layer
   visibility: //boolean representing if layer is visible
   name: //friendly name of the layer
    }
mapControllermapController[]
  • mapController object
onVisiblestring
  • A callback function when user turns on the layer
onHiddenstring
  • A callback function when user turns off the layer
onOpacityChangestring
  • A callback function when user changes the opacity of the layer
