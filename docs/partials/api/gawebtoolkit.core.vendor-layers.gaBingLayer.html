gaBingLayer
directive in module gawebtoolkit.core

Description

Overview

gaGoogleLayer directive is used to create a Bing map.

Usage

as element:
<ga-bing-layer
       layer-type="{string|@}"
       bing-api-key="{string|@}"
       wrap-date-line="{string|@}"
       visibility="{string|@}">
</ga-bing-layer>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
layerTypestring@
  • Required. Specified Bing maps layer type. Eg, Road.
bingApiKeystring@
  • Required. Your own Bing maps API key.
wrapDateLinestring@
  • A boolean value ('true', 'false') which defines the map in the layer should be wrapped or not. If wrapped then the map will be unlimited scrollable.
visibilitystring@
  • A boolean value ('true', 'false') which enables or disables visibility of the layer.
