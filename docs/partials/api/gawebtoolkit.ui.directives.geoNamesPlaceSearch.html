geoNamesPlaceSearch
directive in module gawebtoolkit.ui

Description

Simple control exposing google auto complete search which zooms on selection.

Usage

as element:
<geo-names-place-search
       map-controller="{object}"
       search-icon-url="{string}"
       geo-names-api-key="{string}"
       zoom-level="{number}"
       country-code="{string}"
       result-template-url="{string}"
       on-results="{Function}"
       on-results-selected="{Function}"
       on-perform-search="{Function}"
       activate-key="{number}">
</geo-names-place-search>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
mapControllerobject
  • Map controller
searchIconUrlstring
  • Path to an icon used for search
geoNamesApiKeystring
  • Username to use for geonames.org web service
zoomLevelnumber
  • Zoom level after selection from autocomplete
countryCodestring
  • Country code to be used in the search
resultTemplateUrlstring
  • URL for the result template HTML
onResultsFunction
  • function called when results are returned
onResultsSelectedFunction
  • function called when a result is selected
onPerformSearchFunction
  • function called when search is performed
activateKeynumber
  • key code for activating the search
