gaMapMarker
directive in module gawebtoolkit.core

Description

A wrapper for a native map marker

Usage

as element:
<ga-map-marker
       marker-icon="{string|@}"
       marker-lat="{string|@}"
       marker-long="{string|@}"
       marker-width="{string|@}"
       marker-height="{string|@}"
       layer-name="{string|@}">
</ga-map-marker>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
markerIconstring@
  • Marker icon url
markerLatstring@
  • Latitude for the marker
markerLongstring@
  • Longitude for the marker
markerWidthstring@
  • Width set for the marker icon
markerHeightstring@
  • Height set for the marker icon
layerNamestring@
  • A name allocated to the marker
