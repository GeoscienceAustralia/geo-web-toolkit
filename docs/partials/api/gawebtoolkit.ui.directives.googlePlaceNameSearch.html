googlePlaceNameSearch
directive in module gawebtoolkit.ui

Description

Simple control exposing google auto complete search which zooms on selection.

Usage

as element:
<google-place-name-search
       map-controller="{object}"
       search-icon-url="{string}"
       zoom-level="{number}"
       country-code="{string}">
</google-place-name-search>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
mapControllerobject
  • Map controller
searchIconUrlstring
  • Path to an icon used for search
zoomLevelnumber
  • Zoom level after selection from autocomplete
countryCodestring
  • Google country code to be used in the search
