gaLayerOpacitySlider
directive in module gawebtoolkit.ui

Description

Adds an opacity slider to the map and attaches it to selected layer

Usage

as element:
<ga-layer-opacity-slider
       layers-id="{string}"
       layers-opacity="{string}"
       map-controller="{mapController[]}">
</ga-layer-opacity-slider>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
layersIdstring
  • The ID of the layer
layersOpacitystring
  • reference to opacity value of the layer
mapControllermapController[]
  • mapController object
