geoLayersDropDown
directive in module geowebtoolkit.ui

Description

This control displays a select box of layers and on change, notify via event. Used in a restricted group of layers. Used for selecting between a list of mutually exclusive layers.

Usage

as element:
<geo-layers-drop-down
       layers-data="{Layer[]}"
       selected-model="{string}"
       controller-emit-event-name="{string}"
       on-selected-layer-changed="{Function}"
       layer-group-id="{string}">
</geo-layers-drop-down>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
layersDataLayer[]
  • Layers that the control uses to switch layers
selectedModelstring
  • Id of the layer that is currently selected
controllerEmitEventNamestring
  • event name that is fired that passes the controller
onSelectedLayerChangedFunction
  • function that is called when the selected value has changed
layerGroupIdstring
  • A discriminator that is passed to events to identify which group of layers
