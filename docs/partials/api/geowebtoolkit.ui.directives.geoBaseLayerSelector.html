geoBaseLayerSelector
directive in module geowebtoolkit.ui

Description

This control displays a select box of layers and on change, switches base layers between the proivided list.

Usage

as element:
<geo-base-layer-selector
       layers-data="{Layer[]}"
       map-controller="{object}"
       controller-emit-event-name="{string}">
</geo-base-layer-selector>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
layersDataLayer[]
  • Layers that the control uses to switch base layers
mapControllerobject
  • controller instance for the map
controllerEmitEventNamestring
  • event that is fired that passes the controller
